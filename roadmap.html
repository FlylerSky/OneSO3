Firestore: Thiếu quyền đọc config/roadmap
Trang vẫn hoạt động với cấu hình mặc định, nhưng để load config từ Firestore bạn cần thêm rule sau: 
// Firestore Rules
match /config/{docId} {
  allow read: if true;
  allow write: if request.auth != null
    && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.type == 'admin';
}

match /roadmap_leaks/{id} {
  allow read: if true;
  allow write: if request.auth != null
    && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.type == 'admin';
}
Mở Firebase Console → Firestore → Rules Dán rule trên vào và nhấn Publish Tải lại trang này
